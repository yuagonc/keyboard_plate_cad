#### 部分定位板相关知识
键帽两个1U对应轴孔间隔为
1U = 0.75inch = 0.75 * 25.4 = 19.05
其他轴孔间距可由此推算得出
例如1.5U和1U间隔为
(1 / 2 + 1.5 / 2) * 0.75 * 25.4 = 23.8125
